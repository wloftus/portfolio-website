PlugMeIn Android Application

Project Description

PlugMeIn is an Android Application designed to help electric car owners locate and review chargers around Vancouver. My team found that using Google Maps to locate electric chargers could be overwhelming to the user with the amount of information presented. Because of this we decided to create an App specifically for this purpose.

In a team of 4, I worked with my classmates to design this App using technologies such as Java, Android Studio, Firebase (NOSQL Database), and Google's Maps and Static Street View APIs. This App utilized Vancouver's Open Data API to retrieve information on electric car chargers. I had a lot of fun working on this project. It was my first time creating a Mobile App with Android Studio, which was exciting. I also found working with the Google APIs very cool and was amazed at what we could do with them.

What I Did

Some of my technical work included:

  • Designing User Stories and User Flow diagrams
  • Creating wireframes using Figma and implementing them into the App's frontend
  • Performing GET requests to the Vancouver Open Data API and parsing the JSON response
  • Researching and integrating Google's Maps and Static Street View API into the App
  • Using the Open Data API JSON response to create Charger objects and displaying them on the map

Some of my non-technical work included:

  • Maintaining a Trello Board to keep track of tasks for the team
  • Problem solving collaboratively with my team to troubleshoot bugs
  • Working independently and explaining what I accomplished or what problems I ran into during team meetings

Pictures

Wireframes

These are the four Wireframes I created for our App using Figma. I wanted the interface to feel familiar and intuitive for users so I followed popular designs. These Wireframes would be later implemented into the App's frontend by myself.



Sign In Page

This the Application's Sign-in page. Users are taken here when they start the App. We used Firebase's authentication services to allow users to log in with Google, their email, or their phone.



Charger Details Page

This is the page you are taken to after selecting a charger from the map. The charger information is passed to this page in the form of an object and the relevant information is filled in on the screen. The picture you see is obtained using Google's Static Street View API. We can query the API with the Charger's exact coordinates and it will return the closest Street View image. I thought this would be a useful way for the Users to visualize the Charger's location without us having to manually build up a database of Charger pictures.
