Lighter Footprint Suggestions API and CMS

Project Description

Lighter Footprint is a Web Application helped to designed people build better habits-for themselves, and the planet. The App contains a quiz that will calculate your carbon footprint after you take it. The App will then give you suggestions and goals based on your quiz answers, on how to better reduce your carbon footprint.

I worked in a team of 4 for our industry client, Lighter Footprint, to design a containerized backend API for retrieving and creating suggestions. We developed following Agile and used technologies such as GraphQL, TypeScript, Docker, and Strapi (a headless CMS). I really enjoyed working on this project and getting the chance to work with all these new and exciting technologies. The goal of the Lighter Footprint App, which is to reduce our environmental impact, made this the project even more worthwhile.

What I Did

Some of my technical work included:

  • Using Docker to create containerized development, staging, and production environments
  • Working with asynchronous functions and Promises in TypeScript
  • Designing the GraphQL Schema and Query Resolver Function
  • Creating a Provider using TypeScript, to retrieve Suggestion data from Strapi and map it to a Suggestion object
  • Creating automated unit tests using Jest

Some of my non-technical work included:

  • Participating in daily scrums with my teammates and weekly meetings with the client
  • Demo'ing our weekly deliverables to the client
  • Leading code reviews with my teammates
  • Presenting the finished project to our classmates and other industry clients

GraphQL Query Resolver

This is the GraphQL Query Resolver I coded. These GraphQL queries are the end-point between our API and the client's Web Application so they were very important to our project. I designed them based on the Client's requirements and the use cases provided to us. The first query, suggestions(), will retrieve multiple suggestions and filter them based on the parameters provided. The second query, suggestion(), only retrieves a single suggestion by its ID.



Suggestions Provider

The JSONSuggestionsProvider is where our API communicates with Strapi to retrieve suggestions. When the Provider is initialized, it takes advantage of Node.js's process.env.NODE_ENV to determine which Strapi environment to connect to. The getSuggestions() function sends a GET request to Strapi and then calls the handleResponse() function to map the response to a Suggestion object.
