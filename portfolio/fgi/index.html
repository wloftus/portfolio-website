Forensic Guardians International Web Application

Project Description

Forensic Guardians International is a company that provides a biological profiling service. This service, implemented, through the Web App, collects, encrypts, and stores any of the client's identifying features. These could be pictures of the client's tattoos, birthmarks, scars, and so on. These features can then be used to identify clients in the case of an emergency.

I worked in a team of 4 for our industry client, Forensic Guardians International, to design a intuitive UI and backend for storing pictures and meta-data of customer's identifying features. We developed following Agile and used a Scrum Board. During the project I got to use and become proficient in technologies such as React, Node.js Express, and PostgreSQL. My role in the was a Full Stack Developer and I really enjoyed being able to work with and master all aspects of the Web App. Since our client gave us a lot of freedom in terms of our implementation, I found it really exciting to take their requirements and develop our own solutions to them.

What I Did

Some of my technical work included:

Some of my non-technical work included:

GraphQL Query Resolver

This is the GraphQL Query Resolver I coded. These GraphQL queries are the end-point between our API and the client's Web Application so they were very important to our project. I designed them based on the Client's requirements and the use cases provided to us. The first query, suggestions(), will retrieve multiple suggestions and filter them based on the parameters provided. The second query, suggestion(), only retrieves a single suggestion by its ID.



Suggestions Provider

The JSONSuggestionsProvider is where our API communicates with Strapi to retrieve suggestions. When the Provider is initialized, it takes advantage of Node.js's process.env.NODE_ENV to determine which Strapi environment to connect to. The getSuggestions() function sends a GET request to Strapi and then calls the handleResponse() function to map the response to a Suggestion object.
